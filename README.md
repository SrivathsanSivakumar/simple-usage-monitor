# MorePrompts
### CLI Tool - real-time token and cost estimator that lives in your terminal for Claude Code

- Know your token and dollar cost in __real-time__ as your type your prompts, in the same terminal, so you know how many tokens and associated cost your input uses
- Tool seamlessly integrates with Claude Code with no additional setup needed! Just install and run `main.py` from anywhere to start. 
- Your sessions are fully private. MorePrompts does not store or log any data from any sessions.

# Installation
Clone the repo

`git clone https://github.com/SrivathsanSivakumar/MorePrompts`

Then from anywhere, run

`python3 path/to/MorePrompts/main.py`
